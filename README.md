### INSTALL:
```
pyvenv chatbot
```
```
cd chatbot
```
```
source bin/activate
```
```
git clone https://github.com/kirimaks/chatbot.git
```
```
cd chatbot
```
```
pip install --upgrade pip
```
```
pip install -r requirements.txt
```

### RUNNING:
```
python bot.py
```
